Constructs a new Brownian object.

Constructor to create a new Brownian object, generating increments in x and y to represent Brownian motion with a given bias distance, bias angle, Brownian RMS and PSF type.

Parameters
distBias distance for the star movement /arcsec
angleBias angle for star movement /deg, with zero in the x-direction, angles defined anti-clockwise
rmsRMS of Brownian motion distance /arcsec
typeWhether the data source is a Huygens PSF. True for Huygens, false for FFT.
