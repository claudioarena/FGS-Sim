Sample the PSF with given centre coordinates.

The normalised matrix of photons is always centred, so this function samples from it to output a PSF matrix centred at the given coordinates. Some photons are not detected to one side, and some simels are empty on the other side and have 0 values.

Parameters
xCentreSimel x-coordinate of PSF centre
yCentreSimel y-coordinate of PSF centre
Returns
Matrix of photons with the PSF centred at the given coordinates.
